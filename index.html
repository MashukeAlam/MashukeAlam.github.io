SkyDrop

Price Tracker Web

Developed as a group project using NodeJS, the Price Tracker Web is a platform that displays trends in game prices. It utilizes cron jobs to scrape price data and presents it in graphical format, providing users with valuable insights.

StudySense is a desktop application designed to automate login processes for UIU ELMS and UCAM platforms. It checks for the latest material uploads, enhancing user convenience. Note: StudySense is optimized for older versions of UIU's websites.
